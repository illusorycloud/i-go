产品介绍

产品概述

VAPTCHA是“Variation Analysis based Public Turing Test to Tell Computers and Humans Apart”（基于变量分析来区分人类和计算机的图灵测试程序）的缩写，又称为手势验证码，一种基于人工智能和大数据的人机验证解决方案 。用户仅需用鼠标绘制指定轨迹即可完成人机验证。VAPTCHA能有效防止恶意密码破解、论坛灌水、垃圾邮件、撞库等。

产品特性

传统的人机验证解决方案，不是验证码识别困难，就是安全性不高，极易被自动化程序破解。而VAPTCHA的人机验证策略则能同时兼顾便捷性及安全性。

特点 描述
易于识别 不会出现传统验证码的变形严重导致人类不好识别的情况；
不以特定的语言字库作为验证码，任何语言背景的网页环境都能完美部署。
速度快 不同于传统验证码，VAPTCHA仅需要用户用鼠标绘制指定轨迹，在不超过1秒的时间内即可完成人机验证；
那些经过风险评估，安全性高的客户端仅需要单击提交按钮即可完成验证，耗时不过200毫秒。
极安全 通过反机器学习来生成验证图；
轨迹和背景随机组合且轨迹融入背景，具备抽象思维的人脑仍然很容易识别而机器看到的则是毫无逻辑的像素点；
利用机器学习构建庞大的行为特征库来甄别人与自动化程序；
利用设备指纹跟踪可疑客户端并根据历史验证记录进行辅助判断；
实时动态更新的黑名单库，连续跨站攻击者将无所遁形；
经过破解测试，VAPTCHA的安全级别远远超过其他人机验证解决方案。

应用场景

传统的登录注册、投票、订票等需要人机验证的地方都可以用到VAPTCHA，尤其是对安全性要求很高，同时又希望良好用户体验的场景。验证的便利性让VAPTCHA可以应用到PC、移动设备甚至嵌入式设备。
